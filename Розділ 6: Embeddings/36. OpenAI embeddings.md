# OpenAI Embeddings API

У цій лекції ми практично працюємо з **Embeddings API від OpenAI** — генеруємо embeddings через офіційну бібліотеку.

## Проєкт

Новий окремий проєкт (як і попередні в курсі):

- **TypeScript**
- **ts-node**
- бібліотека **OpenAI**

Можна використати інший існуючий проєкт або створити новий — код доступний як ресурс курсу.

---

## Функція `generateEmbeddings`

Потрібні:

1. об'єкт **OpenAI** (клієнт);
2. async-функція, яка приймає `input: string | string[]`.

### Виклик API

```typescript
const response = await openai.embeddings.create({
  input,   // текст або масив рядків
  model: 'text-embedding-3-small',
});
```

### Параметри

| Параметр | Опис |
|----------|------|
| **input** | рядок або масив рядків для вбудовування |
| **model** | модель embeddings |

### Яку модель обрати

Раніше основною була **text-embedding-ada-002** — працювала добре, але дорожча.

Зараз є дві нові моделі серії **text-embedding-3**:

- **text-embedding-3-small**
- **text-embedding-3-large**

У курсі використовуємо **text-embedding-3-small**: за якістю майже як ada-002, але **набагато дешевша** (як обговорювалось у попередній лекції).

## Структура відповіді

```typescript
return response.data;
```

Поле **`data`** — масив об'єктів. Кожен об'єкт містить **`embedding`** — масив чисел.

### Один рядок → один embedding

Якщо передати один рядок (наприклад, `"cat"` або `"The cat is on the roof"`), у **`data`** буде **один** елемент — один embedding на весь текст, незалежно від довжини речення.

### Що всередині embedding

- масив із **~1536 чисел** (вимірів моделі text-embedding-3-small);
- значення зазвичай в діапазоні **від -1 до 1**;
- на початку масиву часто багато значень близьких до нуля.

Приклад запуску:

```bash
npm start
```

У консолі або debug view видно масив чисел — це і є згенерований embedding.

---

## Кілька рядків → кілька embeddings

Якщо передати **масив рядків**:

```typescript
generateEmbeddings([
  'The cat is on the roof',
  'The dog is on the car',
]);
```

У **`response.data`** повертається **масив із двох об'єктів** — по одному embedding на кожен рядок у масиві.

**Правило:** OpenAI генерує **окремий embedding для кожного рядка**, переданого в `input`.

---

## Підсумок

| Input | Результат |
|-------|-----------|
| `string` | 1 embedding у `data[0]` |
| `string[]` | N embeddings — по одному на кожен рядок |

Ми згенерували перший embedding через OpenAI API.
