# Розрахунок схожості (similarity)

Тепер, коли є дані та збережені embeddings, можна **порахувати схожість між ними**.

Створюємо новий файл **`similar.ts`** і реалізуємо **два способи** обчислення схожості між двома наборами embeddings — як у презентації розділу:

1. **Dot product** (скалярний добуток)
2. **Cosine similarity** (косинусна схожість)

---

## Функція `dotProduct`

Приймає **два масиви чисел** однакової довжини і повертає їх **скалярний добуток** — суму попарних добутків відповідних компонентів.

```typescript
function dotProduct(a: number[], b: number[]): number {
  return a.reduce((sum, val, i) => sum + val * b[i], 0);
}
```

---

## Функція `cosineSimilarity`

Також приймає **два масиви чисел** — вони мають бути **однакової довжини** (інакше варто кинути помилку; у лекції цей кейс не реалізують).

Алгоритм:

1. обчислити **dot product** двох векторів;
2. обчислити **magnitude** (довжину) кожного вектора;
3. розділити dot product на добуток magnitudes — отримуємо **кут між двома векторами**.

Cosine similarity показує, наскільки напрямки векторів збігаються, незалежно від їхньої «довжини».

## Рефакторинг `data.ts`

Щоб не дублювати код, з **`data.ts`** експортуємо:

- тип **`DataWithEmbeddings`**;
- функцію **`generateEmbeddings`**;
- функцію **`loadJSONData`**.

## Функція `main` у `similar.ts`

### 1. Завантажити embeddings з файлу

```typescript
const dataWithEmbeddings = loadJSONData<DataWithEmbeddings[]>(
  "dataWithEmbeddings.json"
);
```

Generic **`DataWithEmbeddings[]`** дає TypeScript type safety для подальшої роботи.

### 2. Задати пошуковий запит

```typescript
const input = "animal";
```

Мета — знайти, **які записи в `dataWithEmbeddings.json` найбільше схожі на слово «animal»** за їхніми embeddings.

### 3. Порахувати схожість для кожного запису

Створюємо масив:

```typescript
type Similarity = {
  input: string;
  similarity: number;
};

const similarities: Similarity[] = [];
```

Для кожного `entry` з `dataWithEmbeddings`:

1. генеруємо embedding для запиту:

   ```typescript
   const inputEmbedding = await generateEmbeddings(input);
   // inputEmbedding[0].embedding — embedding для "animal"
   ```

2. обчислюємо схожість (у лекції використовують **cosine similarity**):

   ```typescript
   const similarity = cosineSimilarity(
     entry.embedding,
     inputEmbedding[0].embedding
   );
   ```

3. додаємо результат у масив:

   ```typescript
   similarities.push({
     input: entry.input,
     similarity,
   });
   ```

### 4. Відсортувати і вивести

```typescript
similarities.sort((a, b) => b.similarity - a.similarity);

similarities.forEach((entry) => {
  console.log(entry.input, entry.similarity);
});
```

Сортування за спаданням — **найбільш схожі записи зверху**.

| Крок | Дія |

| 1 | Завантажити збережені embeddings з `dataWithEmbeddings.json` |

Dot product і cosine similarity — **найпоширеніші метрики** для порівняння векторів. У цій лекції для пошуку використовують **cosine similarity**.
