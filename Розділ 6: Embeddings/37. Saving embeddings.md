# Збереження embeddings

У цій лекції ми **генеруємо кілька embeddings з файлу на диску** і **зберігаємо їх у JSON-файл**.

Зазвичай цю роботу виконує **vector database** (векторна база даних). Але в цьому розділі ми робимо все **самостійно**, а в наступній лекції вже підключимо векторну БД і порахуємо схожість між embeddings.

---

## Підготовка даних

Створюємо файл **`data.json`** — простий список рядків. У прикладі згенеровано через ChatGPT три категорії:

- тварини;
- рослини;
- комахи.

```json
[
  "lion",
  "elephant",
  "dolphin",
  "eagle",
  "rabbit",
  "oak",
  "rose",
  "sunflower",
  "fern",
  "cactus",
  "bee",
  "butterfly",
  "ant",
  "dragonfly",
  "ladybug"
]
```

## Допоміжні функції для роботи з файлами

Додаємо дві утиліти на Node.js:

### `loadJSONData`

- **`readFileSync`** з модуля `fs`;
- **`join`** з модуля `path` — коректно збирає шлях до файлу;
- **`JSON.parse`** — перетворює вміст файлу в об'єкт/масив.

Для TypeScript додаємо **generic** до `loadJSONData`, бо `JSON.parse` повертає `any`. З generic можна вказати очікуваний тип і отримати type safety.

### `saveDataToJSONFile`

- **`writeFileSync`** з модуля `fs` — записує дані у JSON-файл на диск.

---

## Функція `main`

Основний потік роботи:

1. **Завантажити дані** з `data.json` — TypeScript знає, що `data` має тип `string[]`.
2. **Згенерувати embeddings** для всіх рядків одним викликом API.
3. **Зібрати результат** у масив об'єктів `{ input, embedding }`.
4. **Зберегти на диск** у `dataWithEmbeddings.json`.
5. **Запустити** `main()` через `npm start`.

---

## Код (`embeddings/src/data.ts`)

```typescript
import OpenAI from "openai";
import { readFileSync, writeFileSync } from "fs";
import { join } from "path";

const openai = new OpenAI()

type DataWithEmbeddings = {
    input: string;
    embedding: number[];
}

async function generateEmbeddings(input: string | string[]) {
    const response = await openai.embeddings.create({
        model: "text-embedding-3-small",
        input,
    });
    return response.data;
}

function loadJSONData<T>(filePath: string): T {
    const path = join(__dirname, filePath);
    const rowData = readFileSync(path);
    return JSON.parse(rowData.toString());
}

function saveDataToJSONFile(data: any, fileName: string) {
    const dataString = JSON.stringify(data);
    const dataBuffer = Buffer.from(dataString);
    const path = join(__dirname, fileName);
    writeFileSync(path, dataBuffer);
    console.log(`Data saved to ${fileName}`);
}

async function main() {
    const data = loadJSONData<string[]>("data.json");
    const embeddings = await generateEmbeddings(data);
    const dataWithEmbeddings: DataWithEmbeddings[] = []

    for (let i = 0; i < data.length; i++) {
        dataWithEmbeddings.push({
            input: data[i],
            embedding: embeddings[i].embedding,
        });
    }
    saveDataToJSONFile(dataWithEmbeddings, "dataWithEmbeddings.json");
}

main();
```

## Результат

Після запуску з'являється файл **`dataWithEmbeddings.json`**. Для кожного рядка з вхідного масиву зберігається запис:

| Поле | Опис |
|------|------|
| **input** | оригінальний текст (наприклад, назва тварини, рослини чи комахи) |
| **embedding** | масив чисел — векторне представлення цього тексту |

VS Code зручно форматує великий JSON, тож embeddings легко переглянути.
