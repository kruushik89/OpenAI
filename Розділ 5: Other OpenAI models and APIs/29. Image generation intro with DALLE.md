# Генерація зображень: вступ до DALL-E

У цьому уроці розглядаємо, які **моделі OpenAI** доступні для генерації зображень і чим вони відрізняються.

## Дві моделі: DALL-E 2 і DALL-E 3

| Модель | Характеристика |
|--------|----------------|
| **DALL-E 2** | Дешевша, проста в використанні; якість і роздільність нижчі |
| **DALL-E 3** | Потужніша; підтримує **HD**, загалом значно краща якість зображень |

На практиці DALL-E 2 часто дає слабші результати, тоді як DALL-E 3 генерує зображення помітно кращої якості.

## Кредити та білінг

При створенні акаунта OpenAI ви отримуєте **безкоштовні кредити** (орієнтовно **$5**).

- **DALL-E 2** — можна використовувати на безкоштовних кредитах, обмежень для курсу немає.
- **DALL-E 3** — для роботи з цією моделлю потрібно **додати кредити** на баланс.

### Як поповнити баланс

1. Відкрити **налаштування акаунта** (Settings).
2. Перейти в розділ **Billing**.
3. Обрати **Add credit to balance**.
4. Поповнити карткою на **$5** або **$10**.

Після цього можна використовувати DALL-E 3 і проходити всі приклади з наступних уроків. Якщо хочете лише спробувати — достатньо DALL-E 2.

## API для генерації зображень

Генерація зображень — це **окремий endpoint** OpenAI API, не чат-модель. У документації це **`images.generate`** з відповідними параметрами для вибору моделі (DALL-E 2 або DALL-E 3) та інших опцій.
