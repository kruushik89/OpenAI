# Огляд API та сервісів OpenAI

У першій частині курсу розглядаються сервіси, які надає **OpenAI** — одна з найпопулярніших AI-компаній і зручна точка входу для початку роботи зі штучним інтелектом.

## Категорії API OpenAI

API OpenAI можна умовно згрупувати в **3–4 категорії**. Найважливіша з них — **текстові API**, саме на них зосереджується цей курс.

### 1. Текстові API (text-to-text)

Основний напрям курсу. Дозволяють:

- створювати чат;
- передавати текстовий ввід і отримувати текстову відповідь.

Типові моделі: **GPT-4** та **GPT-3.5** — це моделі типу *text-to-text*.

Також є **моделі перекладу**, які будуть розглянуті окремо.

### 2. Зображення (Image API)

Моделі генерації зображень за текстовим описом. Найвідоміша — **DALL-E** (назва поєднує ім’я художника Сальвадора Далі та персонажа з мультфільму *Wall-E*).

### 3. Аудіо (Audio models)

Підтримують:

- **text-to-speech** — перетворення тексту в мовлення;
- **speech-to-text** — розпізнавання мовлення в текст (модель **Whisper**).

### 4. Відео (Sora)

Нова можливість OpenAI: текстовий ввід → відео на виході (**Sora**). У цьому курсі **не розглядається**, оскільки технологія ще дуже нова.

## Що буде в курсі

| Розділ | Тема |
|--------|------|
| Наступні два розділи | Текстові API (GPT, чат, переклад) |
| Пізніше (5-й розділ) | Інші API та моделі OpenAI (зображення, аудіо тощо) |

**Підсумок:** OpenAI пропонує широкий набір інструментів — від тексту та перекладу до зображень, аудіо та відео. Курс починає з текстових можливостей, а решту сервісів розглядає далі.
