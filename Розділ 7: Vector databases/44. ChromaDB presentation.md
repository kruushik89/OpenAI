# ChromaDB: перша vector database

Перша vector DB у розділі — **ChromaDB**.

## Що це

- **Open source** vector database
- Написана на **Python**
- Можна запускати через **Python** або **Docker**

## Вимоги

Щоб слідувати за курсом, потрібен **Python** або **Docker**. Автор обирає **Docker**: достатньо **двох Docker-команд**, щоб підняти ChromaDB server.

## Спостереження автора

Документація для **JavaScript** трохи слабка, але нічого надто складного — проблеми можна вирішити.

## Сайт і документація

На сайті ChromaDB позиціонується як **проста**, **lightweight** vector database з базовими vector operations. Усі ці операції розберемо в наступних лекціях розділу.

Якщо щось не працює — варто зайти на **GitHub** проєкту (з сайту) і перевірити **Issues**: часто там уже є рішення.
