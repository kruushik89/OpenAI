# Tool call setup — перший OpenAI tool

У цій лекції починаємо писати **перший OpenAI tool**. Створюємо нову папку `tools` і файл `index.ts` у проєкті.

## Чотири кроки роботи з tools

Коли ми пишемо tools для OpenAI, потрібно пройти **чотири кроки**:

1. **Налаштувати chat tools** і зробити **перший виклик OpenAI**.
2. **Проаналізувати відповідь** і вирішити, чи потрібно викликати tool (function).
3. **Викликати tool** — виконати відповідну функцію.
4. Зробити **другий виклик OpenAI** з результатом tool.

> У лекції автор часто вживає слова *tools* і *functions* як синоніми. У старішій документації OpenAI API вони називалися **functions** — по суті це те саме.

## Підготовка коду (крок 1 — без tools)

На цьому етапі реалізуємо лише **перший крок** — перший виклик API, ще **без** налаштування tools.

### Імпорт і клієнт

```ts
import OpenAI from "openai";

const client = new OpenAI();
```

### Асинхронна функція `callOpenAIWithTools`

Логіку обгортаємо в `async function callOpenAIWithTools()`.

### Контекст (messages)

Масив типу `ChatCompletionMessageParam[]`:

| Роль | Зміст |
|------|-------|
| `system` | *You are a helpful assistant that gives information about the time of the day.* |
| `user` | *What is the time of the day?* |

Системне повідомлення підказує моделі, що вона має давати інформацію про час доби. Запит користувача задається прямо в коді (без CLI, як у попередньому розділі).

### Перший виклик API

```ts
const response = await client.chat.completions.create({
  model: "gpt-4o-mini",
  messages: context,
});
```

Tools поки **не додаємо** — лише перевіряємо поведінку без них.

### Повний код `tools/index.ts`

```ts
import OpenAI from "openai";
import { isContext } from "vm";

const client = new OpenAI();

async function callOpenAIWithTools() {
  const context: OpenAI.Chat.ChatCompletionMessageParam[] = [
    {
      role: "system",
      content:
        "You are a helpful assistant that gives information about the time of the day.",
    },
    {
      role: "user",
      content: "What is the time of the day?",
    },
  ];
  const response = await client.chat.completions.create({
    model: "gpt-4o-mini",
    messages: context,
  });
  console.log(response.choices[0].message.content);
}

callOpenAIWithTools();
```

### Запуск

- Шлях у `package.json` змінюється на `tools/index.ts`.
- Запуск: `npm start`.
- Відповідь виводимо в консоль.

## Результат без tools

Модель відповідає, що **не може надати інформацію в реальному часі** — саме так і очікується.

Це демонструє проблему, яку вирішує **function/tool call**: без зовнішньої функції модель не знає поточного часу.

## Що далі

У наступній лекції — **налаштування function call**, щоб модель могла отримувати актуальний час через наш tool.
