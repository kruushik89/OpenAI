# First tool call — перший виклик function/tool

У цій лекції завершуємо налаштування **першого tool call**: додаємо tool до запиту, обробляємо відповідь API, викликаємо функцію та робимо **другий запит** до OpenAI з результатом.

Це кроки **2–4** з чотирьох, описаних у попередній лекції:

1. ~~Налаштувати chat tools і зробити перший виклик~~ (було в лекції 23)
2. **Проаналізувати відповідь** і вирішити, чи потрібен tool call
3. **Викликати tool**
4. **Другий виклик OpenAI** з результатом tool

---

## 1. Функція `getTimeOfDay`

Спочатку пишемо просту «заглушку» — функцію, яку пізніше замінимо реальною логікою:

```ts
function getTimeOfDay(): string {
  return new Date().toLocaleTimeString();
}
```

---

## 2. Поле `tools` у запиті

У `openAI.chat.completions.create()` додаємо масив `tools` (може містити кілька функцій; тут — одну):

```ts
tools: [
  {
    type: "function",
    function: {
      name: "getTimeOfDay",           // має точно збігатися з ім'ям функції в коді
      description: "Get the current time of the day",
    },
  },
],
```

- `type: "function"` — тип tool.
- `function.name` — **точна** назва функції, яку модель має викликати.
- `function.description` — допомагає моделі зрозуміти, коли використовувати цей tool.

---

## 3. Поле `tool_choice: "auto"`

```ts
tool_choice: "auto",
```

`auto` означає: модель **сама вирішує**, чи потрібно викликати tool і який саме (коли tools буде кілька).

---

## 4. Перевірка: чи потрібен tool call?

Після першого відповіді API аналізуємо `finish_reason`:

```ts
const willInvokeFunction =
  response.choices[0].finish_reason === "tool_calls";
```

Можливі значення `finish_reason`: `stop`, `length`, `tool_calls` (старе `function_call` — застаріле).

Якщо `finish_reason === "tool_calls"` — модель хоче викликати функцію.

---

## 5. Отримання даних про tool call

```ts
const toolCall = response.choices[0].message.tool_calls![0];
```

У відповіді (перевірено в debug):

- `finish_reason` → `"tool_calls"`
- `message.tool_calls` → масив викликів
- `tool_calls[0].function.name` → `"getTimeOfDay"`

---

## 6. Виклик функції

```ts
if (willInvokeFunction && toolCall.type === "function") {
  const toolName = toolCall.function.name;

  if (toolName === "getTimeOfDay") {
    const toolResponse = getTimeOfDay();
    // ...
  }
}
```

---

## 7. Додавання результату в контекст

Щоб OpenAI не повернув помилку, у `context` потрібно додати **два** повідомлення:

1. **Повідомлення асистента** з tool call (з першої відповіді):

```ts
context.push(response.choices[0].message);
```

2. **Відповідь tool** з типом `tool`:

```ts
context.push({
  role: "tool",
  content: toolResponse,
  tool_call_id: toolCall.id,
});
```

`tool_call_id` має збігатися з `id` з `toolCall` — це зв'язує результат функції з конкретним викликом.

---

## 8. Другий виклик OpenAI

```ts
const secondResponse = await openAI.chat.completions.create({
  model: "gpt-4o-mini",
  messages: context,
});

console.log(secondResponse.choices[0].message.content);
```

Модель отримує оновлений контекст (system, user, assistant + tool call, tool response) і формує **фінальну** відповідь користувачу.

---

## 9. Повний код `tools/index.ts`

```ts
import OpenAI from "openai";

const openAI = new OpenAI();

function getTimeOfDay(): string {
  return new Date().toLocaleTimeString();
}

async function callOpenAIWithTools() {
  const context: OpenAI.Chat.ChatCompletionMessageParam[] = [
    {
      role: "system",
      content:
        "You are a helpful assistant that gives information about the time of the day.",
    },
    {
      role: "user",
      content: "What is the time of the day?",
    },
  ];
  const response = await openAI.chat.completions.create({
    model: "gpt-4o-mini",
    messages: context,
    tools: [
      {
        type: "function",
        function: {
          name: "getTimeOfDay",
          description: "Get the current time of the day",
        },
      },
    ],
    tool_choice: "auto",
  });

  const willInvokeFunction = response.choices[0].finish_reason === "tool_calls";
  const toolCall = response.choices[0].message.tool_calls![0];

  if (willInvokeFunction && toolCall.type === "function") {
    const toolName = toolCall.function.name;

    if (toolName === "getTimeOfDay") {
      const toolResponse = getTimeOfDay();
      context.push(response.choices[0].message);
      context.push({
        role: "tool",
        content: toolResponse,
        tool_call_id: toolCall.id,
      });
    }
  }

  const secondResponse = await openAI.chat.completions.create({
    model: "gpt-4o-mini",
    messages: context,
  });

  console.log(secondResponse.choices[0].message.content);
}

callOpenAIWithTools();
```

---

## Результат

Після `npm start` асистент отримує **реальний** час через `getTimeOfDay()` і **інтерпретує** його: наприклад, що зараз вечір або ранній вечір. Модель додає власний «стиль» відповіді, бо в system prompt вона — helpful assistant про час доби.
